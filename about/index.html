我是「多小凯」，这个网名是我在公司的花名经过些微调整而来。真名是「肖嘉兴」。

由于初高中比较爱上网，大学就选择了软件工程专业。当时觉得 HTML、CSS 能画出任意想象的页面，其他如 C/C++、Java、数据库等开发随便弄弄能跑就行… too young，too simple，sometimes naive。

刚学 JavaScript 那会喜欢实现以前玩过的各种小游戏，如 别踩白块儿、飞机大战、坦克大战等。到了工作后学习了 React 后写了 超级魂斗罗，学了 Three.js 后写了 微信跳一跳。这些都只是 Demo 练习级别 🤡。

目前在国内电商 Top3 的一家公司做前端开发，之前在 Top3 的另外一家，再早之前是一个小而美的跨境电商，其他就没有了，工作地一直都在上海。

工作前两年(2017年～2019年)主要是做可视化页面搭建系统开发，最近四年(2019年～至今)主要是服务于前端开发，又名前端首席客服。作为 🐛 Bug 中区枢纽需要一直处理各种问题，为了避免忘记而重复处理相同的问题所以把过往的问题都记录在了该站点。如果这些文章刚好对你有帮助，我会很开心。

关于未来的生活与工作总之开心最重要～
