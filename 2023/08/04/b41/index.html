2023-08-04
C++

иғҢжҷҜ

жңҖиҝ‘еҮәдәҺйңҖжұӮйңҖиҰҒ, еӯҰд№ дәҶдёҖдёӢеҫ®дҝЎйҖҶеҗ‘ wxhelper зҡ„д»Јз Ғе®һзҺ°, wxhelper жҡҙйңІдәҶеӨ§йҮҸзҡ„еҫ®дҝЎжҺҘеҸЈ, дҪҝеҫ—иғҪеӨҹиҪ»жҳ“еҹәдәҺжӯӨе®һзҺ°дёҖдёӘеҫ®дҝЎжңәеҷЁдәә рҹӨ–пёҸгҖӮеӣ дёәжҳҜз¬¬дёҖж¬ЎжҺҘи§ҰйҖҶеҗ‘зӣёе…іжҠҖжңҜ, еҮәдәҺеҘҪеҘҮе°ұи®°еҪ•дёҖдёӢ

е®һзҺ°еҺҹзҗҶ

еҰӮдёҠзҡ„жөҒзЁӢеӣҫ

  1. йҰ–е…ҲйңҖиҰҒйҖҡиҝҮдёҖдәӣжүӢж®өиҺ·еҸ–еҲ°еҫ®дҝЎдёӯйңҖиҰҒиў«жҡҙйңІзҡ„е…ій”® Call, дҪңиҖ…жҠҠиҮӘе·ұеёёз”Ёзҡ„ж–№жі•и®°еҪ•еңЁ wxhelper/wiki ж–ҮжЎЈ, ж„ҹе…ҙи¶ЈеҸҜд»ҘжҹҘйҳ…
  2. зј–иҜ‘ wxhelper dll зЁӢеәҸжіЁе…ҘеҲ°еҫ®дҝЎиҝӣзЁӢ
  3. иҜҘ dll дјҡеҲӣе»әдёҖдёӘз«ҜеҸЈдёә 19088 зҡ„ http server еҗ‘еӨ–жҸҗдҫӣжҺҘеҸЈжқҘејҖж”ҫи°ғз”Ёе…ій”® Call зҡ„иғҪеҠӣ

дёӢйқўи®ІдёҖдёӢиҜҘ wxhelper dll еҰӮдҪ•е®һзҺ° hook еҫ®дҝЎзҡ„е…ій”® Call, еҰӮдёӢ dll е…ҘеҸЈеҮҪж•°дё»иҰҒжҳҜи°ғз”ЁдәҶ GlobalContext::GetInstance().initialize еҮҪж•°

initialize еҮҪж•°дё»иҰҒжҳҜеҗҜеҠЁдәҶдёҖдёӘ http server

http server зҡ„иҜ·жұӮеӨ„зҗҶеҮҪж•° HttpDispatch еҚіиЎЁзӨәдәҶеҗ‘еӨ–жҸҗдҫӣзҡ„ hook еҫ®дҝЎе…ій”® Call зҡ„е®һзҺ°гҖӮд»Ҙ /api/hookSyncMsg и·Ҝеҫ„зҡ„иҜ·жұӮдёәдҫӢ, еҰӮжһңе®ўжҲ·з«ҜжғійҖҡиҝҮи°ғз”ЁиҜҘжҺҘеҸЈжқҘзӣ‘еҗ¬еҫ®дҝЎж”¶еҲ°ж¶ҲжҒҜзҡ„дәӢд»¶

ж”¶еҲ°иҜҘиҜ·жұӮеҗҺж ёеҝғжҳҜи°ғз”ЁдәҶ wxhelper::hooks::HookSyncMsg еҮҪж•°гҖӮеҸҜд»ҘеҸ‘зҺ° HookSyncMsg еҮҪж•°дёӯеҮәзҺ°дәҶе…ій”®зҡ„DetourAttach зі»з»ҹи°ғз”Ё

DetourAttach жҳҜ Windows е№іеҸ°дёҠзҡ„дёҖдёӘеҮҪж•°пјҢе®ғжҳҜеҫ®иҪҜз ”з©¶йҷўејҖеҸ‘зҡ„ Detours еә“дёӯзҡ„дёҖйғЁеҲҶгҖӮDetours еә“жҸҗдҫӣдәҶдёҖз§Қз”ЁдәҺдҝ®ж”№е’ҢйҮҚе®ҡеҗ‘еҮҪж•°и°ғз”Ёзҡ„жҠҖжңҜпјҢеёёз”ЁдәҺе®һзҺ°еҮҪж•°зҡ„й’©еӯҗе’ҢеҮҪж•°зҡ„еҠЁжҖҒдҝ®ж”№гҖӮ
DetourAttach еҮҪж•°зҡ„дҪңз”ЁжҳҜе°ҶдёҖдёӘзӣ®ж ҮеҮҪж•°зҡ„ең°еқҖдёҺдёҖдёӘиҮӘе®ҡд№үзҡ„еӣһи°ғеҮҪж•°е…іиҒ”иө·жқҘпјҢд»ҺиҖҢеңЁзӣ®ж ҮеҮҪж•°иў«и°ғз”Ёж—¶жү§иЎҢеӣһи°ғеҮҪж•°гҖӮе…·дҪ“жқҘиҜҙпјҢDetourAttach еҮҪж•°дјҡдҝ®ж”№зӣ®ж ҮеҮҪж•°зҡ„д»Јз ҒпјҢдҪҝе…¶еңЁжү§иЎҢеүҚи°ғз”ЁжҢҮе®ҡзҡ„еӣһи°ғеҮҪж•°гҖӮDetourAttach еҮҪж•°зҡ„еҺҹеһӢеҰӮдёӢ:

зҺ°еңЁжҖ»з®—жҳҺзҷҪдәҶ, еҺҹжқҘжҳҜйҖҡиҝҮ DetourAttach зі»з»ҹи°ғз”ЁжҠҠеҫ®дҝЎж”¶еҲ°ж¶ҲжҒҜзҡ„еӨ„зҗҶеҮҪж•° R_DoAddMsg з»ҷд»ЈзҗҶдәҶ, д№ҹе°ұжҳҜжҠҠ R_DoAddMsg и°ғз”ЁйҮҚе®ҡеҗ‘дёәдәҶ HandleSyncMsg, иҖҢ HandleSyncMsg еҮҪж•°йҮҢе°ұеҸҜд»ҘеҶҷдҪ зҡ„дёҡеҠЎйҖ»иҫ‘, жңҖеҗҺ HandleSyncMsg зҡ„е°ҫйғЁи°ғз”ЁдёҖдёӢеҫ®дҝЎзҡ„еҺҹе§ӢеӨ„зҗҶеҮҪж•° R_DoAddMsg еҚіеҸҜ

з”Ё JavaScript и§ЈйҮҠ DetourAttach зҡ„иЎҢдёәе°ұжҳҜ

жңҖеҗҺзҡ„дёҖдёӘе…ій”®зӮ№е°ұжҳҜеҰӮдҪ•иҺ·еҸ–еҫ®дҝЎеҺҹжң¬зҡ„еӨ„зҗҶеҮҪж•° R_DoAddMsg зҡ„ең°еқҖ, еҸҜд»ҘзңӢеҲ° R_DoAddMsg ең°еқҖе…¶е®һе°ұжҳҜиҜҘ dll зҡ„ең°еқҖ GetWeChatWinBase() + д№ӢеүҚйҖҡиҝҮйҖҶеҗ‘жүӢж®өиҺ·еҸ–еҲ°зҡ„иҜҘеҮҪж•°зҡ„еҒҸз§»йҮҸ offset::kDoAddMsg

иҺ·еҸ–иҜҘ dll зҡ„ең°еқҖж ёеҝғжҳҜ GetModuleHandleA зі»з»ҹи°ғз”Ё

GetModuleHandleA жҳҜ Windows е№іеҸ°дёҠзҡ„дёҖдёӘеҮҪж•°пјҢз”ЁдәҺиҺ·еҸ–жҢҮе®ҡжЁЎеқ—зҡ„еҸҘжҹ„пјҲhandleпјүгҖӮиҺ·еҸ–еҲ°зҡ„жЁЎеқ—еҸҘжҹ„еҸҜд»Ҙз”ЁдәҺеҗҺз»ӯзҡ„ж“ҚдҪңпјҢдҫӢеҰӮиҺ·еҸ–жЁЎеқ—дёӯзҡ„еҮҪж•°ең°еқҖгҖҒдҝ®ж”№жЁЎеқ—дёӯзҡ„ж•°жҚ®зӯүгҖӮеҮҪж•°еҺҹеһӢеҰӮдёӢ:
